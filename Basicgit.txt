what is git?
Git is a distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
It enable developers to track changes in source
Git records history of changes, allowing you to revert to previous states.
each commit in git represents a recorded state of the project.

Git supports the use of remote repositories hosted on services such as github,gitlab.
integrating git with platforms that support pull requests and code review enhances code quality.
